## DS1 — HIGH: Fix `import config` bare import that silently breaks daily cap checks

**Files to modify:**
- `agents/ruppert/data_scientist/data_agent.py`
- `agents/ruppert/data_scientist/capital.py`

**Background:**
Both files do a bare `import config` (or `import config as cfg`) inside a function.
`config.py` lives at `environments/demo/config.py` — not on `sys.path` — so this raises
`ModuleNotFoundError`, which is silently swallowed by the surrounding `try/except`.
Result: `check_daily_cap_violations()` always returns `[]`. The established pattern in
this codebase is `env_config.get_env_root()` for path resolution; use `importlib` for
clarity and to avoid `sys.path` pollution.

**Change:**

### data_agent.py — `check_daily_cap_violations()`

Replace the bare `import config as cfg` inside the function:

```python

    try:
        from agents.ruppert.data_scientist.capital import get_capital
        import config as cfg
        capital = get_capital()
    except Exception:
        return []
```

```python
# AFTER:
def check_daily_cap_violations(trades_today: list[dict]) -> list[dict]:
    try:
        import importlib.util as _ilu
        from agents.ruppert.data_scientist.capital import get_capital
        from agents.ruppert.env_config import get_env_root as _get_env_root

        _cfg_path = _get_env_root() / 'config.py'
        if not _cfg_path.exists():
            return []
        _spec = _ilu.spec_from_file_location('config', _cfg_path)
        cfg = _ilu.module_from_spec(_spec)
        _spec.loader.exec_module(cfg)

        capital = get_capital()
    except Exception:
        return []
```

### DS3b — `/api/scout/geo`: Remove per-entry Kalshi REST enrichment

```python
                resp = req.get(
                    f"https://api.elections.kalshi.com/trade-api/v2/markets/{e['ticker']}",
                    timeout=3
                )

                    m = resp.json().get('market', {})
                    e['rules']      = m.get('rules_primary', '')
                    e['kalshi_url'] = f"https://kalshi.com/markets/{e['ticker']}"
            except Exception:
                pass

    return entries
```

```python
# AFTER:
def get_geo_scout():
    """Geopolitical markets — cache only, no live REST enrichment.
    rules field is populated by the geo scanner at scan time; if missing here,
    rules_available=False signals the frontend to display a placeholder.
    """
    entries = read_geo_log()
    for e in entries:
        if not e.get('kalshi_url'):
            e['kalshi_url'] = f"https://kalshi.com/markets/{e.get('ticker','')}"
        e['rules_available'] = bool(e.get('rules'))
    return entries
```
